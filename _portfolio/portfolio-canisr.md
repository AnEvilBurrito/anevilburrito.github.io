**CANISR** is a full-stack, production-ready platform designed to standardise and visualise drug response predictive modelling across multi-scale biological datasets.

### Key Features
*   **Full-Stack Architecture**: Built using Postgres, FastAPI, and Next.js.
*   **Analytics Engine**: High-performance engine utilising SciPy for automated percentile ranking and statistical aggregation of model performance (Pearson, R², MSE).
*   **Scalable Backend**: Engineered with PostgreSQL and Redis caching for efficient ML-evaluation retrieval.
